# Boids

An implementation of [Craig Reynolds boids simulation](https://www.red3d.com/cwr/boids/) written in Odin rendered in raylib

## Quick Start

1. Install Odin on your computer
2. Navigate to src directory
   `cd path/to/src`
3. Run the program
   `odin run simulation`
