# Boids

An implementation of (Craig Reynolds boids simulation)[https://www.red3d.com/cwr/boids/] written in Odin

## Quick Start

1. Install Odin on your computer
2. Navigate to project directory
   `cd path/to/project`
3. Run the program
   `odin run .`
